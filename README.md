# fold

> On-chain transaction ledger for Web3 players.

把你的链上交易记录变成看得懂的月度账单。

## Features

- 🔗 Connect any EVM wallet (MetaMask / WalletConnect)
- 🏷 Auto-categorize transactions (Swap / NFT / Airdrop / Gas...)
- 📊 Monthly summary with PnL breakdown
- 🤖 AI-powered on-chain behavior analysis
- 🔒 Local-first, no server, no signup

## Tech Stack

Next.js 14 · WalletConnect · Alchemy API · IndexedDB · Claude AI
